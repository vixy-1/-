GitHub Pages 公開ガイド STATIC HTML


  

    
🚀 このファイル1つで公開できます


    

      
1

      

        GitHubで新しいリポジトリを作成

        github.com → New repository

        名前例: rust-obfuscator / Public を選択
      

    


    

      
2

      

        このファイルをアップロード

        ダウンロードした index.html をリポジトリの main ブランチのルートに配置

        リポジトリ → Add file → Upload files → index.html をドロップ
      

    


    

      
3

      

        GitHub Pages を有効化

        Settings → Pages → Source: Deploy from a branch
Branch: main / (root) → Save
        数分後に公開されます
      

    


    

      
4

      

        公開URLでアクセス

        https://<あなたのユーザー名>.github.io/<リポジトリ名>/
      

    


    

      ✓ 外部依存: Google Fonts のみ（オフライン環境では代替フォントで動作）

      ✓ ビルド不要・npm 不要・サーバー不要

      ✓ AES-GCM は HTTPS 環境（GitHub Pages）で正常動作します
    

  


  

    
⚡ Git CLI でデプロイ

    
git init
git add index.html
git commit -m "feat: RUST OBFUSCATOR v4.0"
git branch -M main
git remote add origin https://github.com/<user>/<repo>.git
git push -u origin main

    
あとは GitHub Settings → Pages から有効化するだけ
